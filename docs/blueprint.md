# **App Name**: Container Solutions Hub

## Core Features:

- Header Navigation: Display a clear and accessible header with navigation menu for easy access to different sections of the website.
- Container Types Showcase: Dedicated page showcasing different container types with detailed specifications and images.
- Product Showcase: Dedicated page for displaying the types of products offered by the container factory with relevant information and visuals.
- Page Navigation: Implement seamless navigation between different pages to enhance user experience.
- Multilingual Support: Support for both Uzbek and Russian languages, allowing users to switch between languages easily.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to convey trust and professionalism.
- Background color: Light gray (#F0F0F0) for a clean, industrial aesthetic.
- Accent color: Steel blue (#548CA8) for interactive elements and highlights, providing a modern feel.
- Body and headline font: 'Inter' sans-serif, which provides a modern and neutral feel for both headlines and body text.
- Use minimalistic icons that reflect industrial themes, such as shipping containers and manufacturing equipment.
- Employ a grid-based layout for organized content presentation.
- Incorporate subtle transitions for a modern, interactive experience.
